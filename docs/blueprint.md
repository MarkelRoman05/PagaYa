# **App Name**: PagaYa

## Core Features:

- User Authentication: Secure user login and registration to access personalized debt tracking. Allows for friend linking based on accounts.
- Friend Management: Allow users to add, manage, and remove friends within the application to facilitate debt tracking.
- Debt Creation: Enable users to create new debt entries, specifying the amount, a description (e.g., 'Cena,' 'Transporte'), and the friend involved.
- Debt Overview: Display a dashboard showing all outstanding debts, categorized by 'Owed to Me' and 'Owed by Me,' with friend details and amounts.
- Debt Settlement: Allow users to mark debts as paid, automatically updating the status for both parties.
- Reminder Notifications: Configure customizable notifications for specific debts, allowing users to send friendly reminders to friends.
- AI-Powered Reminder Message Assistant: An AI tool that suggests polite and personalized reminder messages based on the debt's details and the friend's name, to help draft gentle nudges.

## Style Guidelines:

- Primary color: A balanced and trustworthy cyan-blue (#1D89B8) to reflect clarity and reliability in managing payments. This vibrant tone works well with the light scheme, offering good contrast without being overly dominant.
- Background color: A very light, almost white, cool blue (#EBF3F7) providing a clean and open canvas, aligning with a light visual scheme for a fresh feel.
- Accent color: A fresh and positive teal-green (#2ACFA8) for actions like 'paid' or positive confirmations, chosen to be analogous to the primary while offering significant visual distinction.
- Headline and body text font: 'Inter' (sans-serif) for its modern, highly legible, and objective aesthetic, ensuring clear communication of financial details across devices.
- Minimalist and clear outline icons will be used for actions (e.g., add debt, mark as paid) and navigation, maintaining a modern and user-friendly interface.
- A clean, responsive layout designed for both web and mobile experiences, using card-based UI elements for easy scanning of debt information and actions.
- Subtle and functional animations for feedback, such as confirmation when a debt is settled or when a new debt is added, enhancing user experience without being distracting.
